==異常機制Exception, Error==

工作中程序如果存在各種條件不滿足造成報錯時稱為異常. 所以要考慮各種所有異常情況來防止程序崩潰。 例如Copy要文件存在，目標文件夾存在，空間是否足夠，IO流中途中斷。 邏輯代碼和錯誤處理不應混合地寫，太複雜，JAVA中用TryCatch異常類來處理。

try{JAVA語句}
catch(Exception e){
	e.printStackTrace();
}

java.lang.Throwable 拋出錯誤分為兩種：

  1. Error（JVM崩潰，重啟解決）
  2. Exception（程序異常）: RuntimeException：增加邏輯來規避讓程序執行 CheckedException ：捕獲異常,先捕獲子類 ；
解決方法：
  1. try{}–>catch{}–>finally(回收資源);
  2. throws Exception
